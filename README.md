# xa-components

> 基于Vuejs 2.0的常用组件库

## Document

[在线文档](https://xairFE.github.io/xa-components/docs)

## Install

```bash
npm install xa-components -save
```

## Get Started

```javascript
import Vue from 'vue'
import XaComponents from 'xa-components'
import 'xa-components/dist/xa-components.css'
Vue.use(XaComponents)
```
